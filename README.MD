# Rick and Morty

iOS приложение для справочника Рик и Морти, предоставляющее справочную информацию о персонажах, локациях и эпизодах.

## Основные возможности

- **Персонажи** — каталог персонажей с детальной информацией

Проект использует архитектуру **Clean Swift (ViewController-Interactor-Presenter)**:

## Технологии
